EXAMPLE TRAINING ENTRY
================================================================================

Input Message Format (ChatML):
{
  "messages": [
    {
      "role": "system",
      "content": "Sei il BRAIN di ELAB Tutor, un simulatore educativo..."
    },
    {
      "role": "user",
      "content": "tab: simulator\nesperimento: v1-cap6-esp1\n...\n\nCostruisci un semaforo con 3 LED e un pulsante"
    },
    {
      "role": "assistant",
      "content": "{\"intent\": \"circuit\", \"entities\": [\"led_rosso\", \"led_giallo\", \"led_verde\", \"button1\"], \"tool_calls\": [{\"name\": \"component.add\", \"arguments\": {\"type\": \"led\"}}, ...], \"needs_llm\": true, \"llm_hint\": \"Guida l'utente a costruire il semaforo\"}"
    }
  ]
}

Assistant Response Fields:
  - intent: One of 8 categories
  - entities: Extracted components/experiments
  - tool_calls: Array of {name, arguments} objects
  - needs_llm: Boolean (true if requires LLM reasoning)
  - response: Direct response to user (if needs_llm=false)
  - llm_hint: Guidance for LLM processing (if needs_llm=true)
