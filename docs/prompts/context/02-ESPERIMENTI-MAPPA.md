## Build Modes

Ogni esperimento supporta 3 modalita:
1. **Gia Montato** — componenti posizionati nella posizione finale del libro
2. **Passo Passo** — step-by-step con piazzamento progressivo (posizione finale)
3. **Costruisci** — breadboard vuota, l'utente costruisce
