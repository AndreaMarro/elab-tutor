# Fumetto Report MVP

## Overview

Fine-sessione comic-style report for ELAB lessons. Shows 6 vignettes (3 completed experiments + 3 placeholder "Prossimo esperimento" slots) with UNLIM-generated narrations in Principio Zero v3 tone ("Ragazzi, ..."). Exportable as PDF via browser native print API.

## Use cases

1. **Teacher share report with class**: project on LIM at end of session, kids see recap, then download PDF to share with parents
2. **Teacher archive session**: save PDF to learning records
3. **Kids reflection**: narrations in Principio Zero v3 help teacher verbally narrate recap to class
4. **Next lesson continuity**: placeholder vignettes preview what's coming next

## Architecture

```
LavagnaShell (pending Phase 1.5 wire-up)
    │
    │ user clicks "Fine sessione"
    ▼
SessionReportComic
    ├─ header (ELAB title + date + classroom alias + export button)
    ├─ grid 3×2 (responsive: 3-col / 2-col / 1-col)
    │   ├─ figure × 6
    │   │   ├─ img (or placeholder gradient if no photo)
    │   │   └─ figcaption (title + narration text)
    │   │
    └─ footer ("Grazie ragazzi! Alla prossima lezione!")
```

```js
  onExport={() => { /* custom handler; defaults to window.print() */ }}

```

## Principio Zero v3 compliance

- Narrations MUST use plurale "Ragazzi" (required_any_of pattern)
- Narrations MUST NOT contain "Docente, leggi" or "Insegnante, leggi" (forbidden patterns)
- Enforced via:
  - Unit test assertions (`SessionReportComic.test.jsx` tests 5 + 6)
  - Watchdog CI monitor (`docs/audits/errors-log-YYYY-MM.md` if drift detected live)

Narrations are generated by UNLIM backend (`supabase/functions/unlim-chat` with `[RIFERIMENTO LIBRO FISICO]` tag — see `supabase/functions/_shared/system-prompt.ts`).

## Accessibility (WCAG AA)

- Touch target export button: 44px min-height
- Focus ring: 2px solid Orange #E8941C outline, 2px offset
- Alt text: every `<img>` has descriptive alt
- aria-label: article + button
- Contrast: Navy #1E4D8C on white = 7.4:1 (exceeds 4.5:1 text requirement)
- Reduced motion: @media prefers-reduced-motion disables transitions

## Responsive

- Desktop (>720px): 3-column grid
- Tablet (481-720px): 2-column grid
- Mobile (≤480px): 1-column grid

## Print

- @media print: hides export button, removes padding, page-break-inside on vignettes

## Honest limitations

1. **Photos missing**: fallback gradient placeholders (TRES JOLIE import pending Phase 2)
2. **Not wired to LavagnaShell**: "Fine sessione" trigger pending Phase 1.5 PR
3. **Narrations externally generated**: caller responsible for UNLIM API call and passing narrations prop
4. **Max 6 vignettes**: extras trimmed, no pagination (MVP simplicity)
5. **PDF quality browser-native**: html2pdf.js upgrade pending Phase 2
6. **No share-to-parents URL**: export-only, no email/Slack/Telegram integration (Phase 3)
