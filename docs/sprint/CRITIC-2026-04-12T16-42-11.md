### [NUOVO — ALTO] Back browser da lavagna ancora rotto (bug GF07)
**File**: `src/App.jsx:115-130`

Il popstate handler esiste (`commit C11`) ma ha un bug logico:
```js
function syncFromUrl() {
    const page = getPageFromHash();
    if (page && page !== currentPage) {  // ← BUG: se page è null, non fa nulla
        setCurrentPage(page);
    }
}
```

**Scenario reale**:
1. Utente arriva su `https://www.elabtutor.school/` (no hash) → initialPage='showcase' → pushState NON chiamato
2. Utente inserisce chiave → `navigate('lavagna')` → pushState `/#lavagna`
3. Utente preme Back → URL torna a `/` (no hash) → `getPageFromHash()` → null
4. `if (null && null !== 'lavagna')` → **false** → pagina rimane su LavagnaShell
5. L'utente vede ancora la Lavagna invece della WelcomePage

**Fix**:
```js
const page = getPageFromHash() || 'showcase'; // fallback a homepage
if (page !== currentPage) {
    setCurrentPage(page);
}
```
10 minuti.
