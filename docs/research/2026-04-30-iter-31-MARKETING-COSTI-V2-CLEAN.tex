\section{GDPR --- POSIZIONE in valutazione (non "clean")}

\subsection*{Stato attuale (onesto)}

Stack orientato a minimizzare trasferimenti extra-UE, con primario EU e fallback US gated/anonymized. \textbf{Non ancora certificabile come "GDPR-clean"} senza:

\begin{itemize}[leftmargin=*]
  \item DPA firmabile (Data Processing Agreement) con Mistral, Cloudflare, Supabase, Vercel
  \item Data retention policy documentata
  \item Logging disattivabile o controllato
  \item Subprocessors list pubblica
  \item Regione effettivamente bloccata (verifica config)
  \item No training sui dati garantito contrattualmente
  \item Trattamento dati minori (Art. 8 GDPR consenso parentale)
  \item Ruoli titolare/responsabile chiaramente definiti
  \item DPIA scolastica completata (Data Protection Impact Assessment)
  \item Audit trail accessi sensibili
\end{itemize}

\begin{itemize}[leftmargin=*]
  \item L1 RAG hybrid Voyage 1881 chunks (Vol1+Vol2+Vol3 + 100 wiki + Glossario Tea 174 termini)
  \item L2 Wiki concepts retrieval (100 documenti)
  \item L3 Glossario Tea (174 termini, merged in L1 retrieval)
  \item L4 Class memory (Supabase \texttt{class\_memory} table sessioni passate)
  \item L5 Vision (Pixtral 12B captureScreenshot input)
  \item L6 LLM-knowledge (chat history last N turns)
  \item L7 Analogia graph (concept-graph.json --- defer iter 25+ se mancante)
\end{itemize}

Fusion: RRF k=60 + top-3 inject in BASE\_PROMPT. Timeout 200ms per layer (fail-fast iter 34).
