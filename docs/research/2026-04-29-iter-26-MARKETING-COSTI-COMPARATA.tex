\section{Sommario esecutivo}
% =====================================================

\textbf{ELAB Tutor} (\href{https://www.elabtutor.school}{elabtutor.school}) è un tutor educativo per
elettronica e Arduino destinato a bambini di 8--14 anni nelle scuole italiane.
Il prodotto unifica software web + kit fisici Omaric + tre volumi cartacei (autore Davide Fagherazzi).
La componente AI \emph{UNLIM} richiede un mix di servizi: LLM (Large Language Model),
VLM (Vision-Language Model), TTS (Text-to-Speech), STT (Speech-to-Text), embedding e image generation.

\subsection*{Stack scelto iter 26 (decisione)}

\begin{itemize}[leftmargin=*]
  \item \textbf{LLM primario}: Mistral routing 65/25/10 \\
  (Mistral Small 3.1 / Mistral Large / Llama 3.3 70B Together AI come fallback)
  \item \textbf{Vision (VLM)}: Mistral Pixtral 12B / Pixtral Large
  \item \textbf{STT (Speech-to-Text)}: Cloudflare Whisper Turbo
  \item \textbf{TTS (Text-to-Speech)}: \texttt{edge-tts} (Microsoft, voce Isabella) come baseline,
        Coqui XTTS-v2 self-host (RunPod sandbox iter 28) come opzione voice-cloning
  \item \textbf{Image generation}: Cloudflare FLUX.1 Schnell
  \item \textbf{Embeddings}: \texttt{mistral-embed} primario, BGE-M3 via Cloudflare in fallback
  \item \textbf{Sandbox dev/bench}: RunPod RTX A6000 / RTX 6000 Ada (on-demand, frugale)
\end{itemize}

\section{Glossario --- termini e aziende}
% =====================================================

\subsection{Termini tecnici}

\begin{description}[leftmargin=2cm,style=nextline]
  \item[LLM (Large Language Model)] Modello linguistico di grandi dimensioni. Esempi: Mistral Large, Llama 3.3 70B, Gemini Pro, GPT-4o. Misurati in parametri (es.\ 70B = 70 miliardi).
  \item[VLM (Vision-Language Model)] Modello che capisce immagini + testo. Esempi: Pixtral 12B, Qwen2.5-VL, GPT-4o.
  \item[TTS (Text-to-Speech)] Sintesi vocale: testo $\rightarrow$ audio. Esempi: ElevenLabs, Cartesia, Coqui XTTS, edge-tts.
  \item[STT (Speech-to-Text)] Riconoscimento vocale: audio $\rightarrow$ testo. Esempi: Whisper, Whisper Turbo, Deepgram.
  \item[RAG (Retrieval Augmented Generation)] Pattern dove l'LLM riceve documenti rilevanti recuperati da un database prima di rispondere. Riduce allucinazioni.
  \item[Embedding] Vettore denso che rappresenta semanticamente un testo. BGE-M3 produce vettori di dimensione 1024.
  \item[BGE-M3] BAAI General Embedding M3: modello di embedding multilingue 567M parametri, MIT license, 100+ lingue (italiano nativo).
  \item[RRF (Reciprocal Rank Fusion)] Algoritmo di fusione di ranking da più sorgenti (BM25 + dense). Formula canonica: $\text{rrf}(d) = \sum_i 1/(k + \text{rank}_i(d))$ con $k=60$ (Cormack et al., SIGIR 2009).
  \item[Token] Unità di testo (~4 caratteri italiani). Pricing LLM è quasi sempre per milione di token.
  \item[Pricing input/output] LLM addebitano separatamente token prompt (input) e token generati (output). Output costa di norma 2--4$\times$ input.
  \item[Self-host] Eseguire il modello su infrastruttura propria (es.\ GPU su RunPod). Contrario di API.
  \item[GGUF] Formato di quantizzazione di modelli (es.\ Q4\_K\_M comprime un 7B da 14\,GB a 4\,GB).
  \item[On-demand vs Spot] On-demand = prezzo fisso, sempre disponibile. Spot/Community = ~50\% più economico, può essere interrotto.
  \item[GDPR / DPA] General Data Protection Regulation EU. DPA = Data Processing Agreement (contratto firmato col vendor per trattare dati personali, obbligatorio per minori).
\end{description}

\subsection{Aziende}

\begin{description}[leftmargin=2cm,style=nextline]
  \item[Mistral AI (FR, Parigi)] Startup AI europea fondata 2023 da ex-DeepMind/Meta. Prodotti: Mistral Small/Large, Pixtral, Codestral, Ministral, mistral-embed.
  \item[Cloudflare (US)] CDN globale + Workers AI (inferenza serverless con regions EU). Pay-per-use, free tier 10k Neurons/giorno.
  \item[RunPod (US)] Cloud GPU on-demand e community (spot). Pricing per millisecondo. Ideale per sandbox / bench.
  \item[Together AI (US)] Serverless LLM hosting specializzato Llama (Meta). Tier dedicato e free tier.
  \item[OVHcloud (FR, Roubaix)] Cloud sovrano EU. AI Endpoints lanciato aprile 2025, 40+ modelli open source, Gravelines (FR) datacenter.
  \item[Scaleway (FR, Parigi)] Cloud sovrano EU di Iliad. Generative APIs + Managed Inference + GPU H100 €2,73/h.
  \item[Aleph Alpha (DE, Heidelberg)] AI sovrano tedesco, modelli Luminous (multilingue native incluso italiano). \textbf{Acquisita da Cohere aprile 2026} per circa \$20\,B (deal in chiusura, governance EU mantenuta).
  \item[Black Forest Labs (DE)] Creatori della famiglia FLUX (FLUX.1 Schnell / Pro), modelli image generation open weight di alta qualità.
  \item[Coqui (DE, Berlino)] Sviluppatori open source di XTTS-v2 (TTS multilingue, voice cloning 3--8s, MPL 2.0).
  \item[ElevenLabs (UK $\rightarrow$ globale)] Leader voice TTS premium. 70+ lingue, voice cloning. Pricing da \$5 a \$1.300+/mese.
  \item[HuggingFace (US/FR)] Hub modelli open source + Inference API gestita. 1\,M+ modelli pubblici.
  \item[Voyage AI (US)] Embedding paid, top MIRACL multilingue. Free tier 200\,M token poi \$0,02--0,06/1M.
  \item[Cartesia (US)] TTS premium concorrente di ElevenLabs, ~5$\times$ più economico (\$0,03/minuto).
\end{description}

\subsection{Note di implementazione}

\begin{itemize}[leftmargin=*]
  \item \textbf{Layer 1+2+7} sono recuperati tramite ricerca ibrida: BM25 (sparse, keyword) + dense embedding (BGE-M3 dim 1024) fusi via RRF $k=60$. Riferimento: \texttt{supabase/functions/\_shared/rag.ts:958 LOC} (iter 12 OR-fallback 2-token threshold).
  \item \textbf{Layer 3} usa la chiave classe (\texttt{class\_key} in \texttt{localStorage}) per recuperare ultime 10 sessioni della stessa classe.
  \item \textbf{Layer 4} è il contesto \emph{vivo} (\texttt{lesson-paths-narrative} + state simulatore + ultima azione).
  \item \textbf{Layer 5+6} sono iniettati direttamente nel \emph{system prompt} come segmenti pre-formattati.
  \item Il context complessivo è limitato a 128k token (Mistral Large) con fallback a 32k per Mistral Small.
\end{itemize}

\textbf{Caveat onesto}: la fusione RRF è documentata e testata; il punteggio retrieval italiano K-12 \emph{non è ancora benchmarkato} con gold-set proprio (iter 22 ha shipped fixture 30 query, esecuzione end-to-end deferita iter 27).

\section{Casi possibili --- 5 scenari}
% =====================================================

\subsection{Caso A --- MAX EU sovereign}

\textbf{Stack}: Mistral Small 3.1 (LLM) + OVHcloud Llama 3.3 (fallback) + Coqui XTTS-v2 self-host (TTS) + Aleph Alpha embedding (DE) + Scaleway H100 sandbox.

\textbf{Pro}: Massima sovranità EU, GDPR-pulito al 100\%, narrativa marketing forte (``AI made in EU per scuole italiane'').

\textbf{Contro}: Aleph Alpha Luminous \$30/M = \emph{30$\times$ più caro} di mistral-embed. Costo a scala è proibitivo.

\textbf{Costo stimato 1.000 classi/mese}: ~€500/mese (~5$\times$ caso B).

\subsection{Caso B --- Hybrid EU+US DPA (vincente per ELAB iter 26)}

\textbf{Stack}: Mistral routing 65/25/10 + Cloudflare (FLUX/Whisper/BGE-M3) + Together AI Llama fallback + Coqui RunPod sandbox.

\textbf{Pro}:
\begin{itemize}
  \item Mistral è EU sovereign (Parigi) per il primario
  \item Cloudflare ha region EU + free tier generoso
  \item Together AI fallback con DPA (US) usato solo \emph{gated} (anonimizzato + log audit)
  \item Costo competitivo
\end{itemize}

\textbf{Contro}: Together AI è US (DPA mitiga ma non elimina trasferimento). edge-tts gray-area TOS.

\textbf{Costo stimato 1.000 classi/mese}: ~€170 (vedi tabella sommario).

\subsection{Caso C --- Cheapest CF-heavy}

\textbf{Stack}: Cloudflare Workers AI per tutto (LLM Llama via Workers AI + FLUX + Whisper + BGE-M3).

\textbf{Pro}: Singolo vendor, free tier 10k Neurons/giorno, region EU.

\textbf{Contro}: Cloudflare LLM ancora limitato (no Mistral, no Pixtral nativo); qualità italiano non benchmarkata K-12.

\textbf{Costo stimato 1.000 classi/mese}: ~€80--120 (free tier coperto fino a ~500 classi).

\subsection{Caso D --- All Scaleway managed}

\textbf{Stack}: Scaleway Generative APIs (€0,20/M) + Managed Inference + GPU H100 dedicata.

\textbf{Pro}: Cloud sovrano FR completo, una sola fattura, datacenter Parigi. Predicibilità costi.

\textbf{Contro}: Catalogo modelli più stretto di Mistral diretto. Managed Inference richiede commit minimo.

\textbf{Costo stimato 1.000 classi/mese}: ~€200.

\subsection{Caso E --- Self-host VPS H100 (>10k classi)}

\textbf{Stack}: VPS dedicato H100 (Scaleway o RunPod reserved) con vLLM Llama 3.1 70B + Coqui + BGE-M3 + Whisper.

\textbf{Costo fisso}: H100 ~€2,73/h $\times$ 24 $\times$ 30 = \textbf{€1.965/mese} fissi.

\textbf{Break-even}: ~10.000 classi attive (a costo variabile €0,17/classe equivalente API).

\textbf{Per ELAB iter 26 (200--2.000 classi target)}: \textbf{NON SCEGLIERE}. Eccessivo.

\subsection{Cosa succede se Mistral free tier rate-limit hits a scala 5k+?}

\end{itemize}

\subsection{Cosa succede se Cloudflare FLUX free tier 10k/giorno superato?}

Cloudflare Workers AI: \textbf{10.000 Neurons/giorno} (300k/mese) free, poi pay-per-Neuron.

A 5.000 classi attive: 5.000 immagini/giorno fattibile dentro free tier. A 10.000+ classi: pagamento ~€20--40/mese stimato.

\textbf{Mitigazione}:
\begin{itemize}
  \item Auto-disabilita FLUX per classi inattive (already implemented via lesson-paths active flag).
  \item Cache aggressiva immagini generate (già supportato in Cloudflare R2).
\end{itemize}

\subsection{Cosa succede se Davide Fagherazzi ratify slip Vol3 V0.9?}

Iter 22-30 PDR §3.3 ha già documentato: \emph{Davide schedule slip = LOW probability, LOW impact} (Vol3 V2 ground truth corretto a 0.92, no longer blocker).

\textbf{Mitigazione}: lesson-paths-narrative refactor è indipendente dal contenuto Davide; può essere lanciato senza Vol3 V0.9.

\subsection{Cosa succede se RunPod budget esaurito sandbox dev?}

Budget RunPod attuale: ~\$11 residui (post iter 5 P3 pod TERMINATED + storage).

\textbf{Mitigazione}:
\begin{itemize}
