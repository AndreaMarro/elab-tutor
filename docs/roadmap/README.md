## Come Usare

Ogni sessione:
1. Leggere il file della FASE corrispondente
2. Leggere `00-STATO-ATTUALE.md` per il contesto corrente
3. Eseguire la checklist punto per punto
4. Validare con Chrome Control (mai agenti paralleli)
5. Aggiornare `00-STATO-ATTUALE.md` con i nuovi score
6. Creare `SXX-deliverables.md` con i risultati
