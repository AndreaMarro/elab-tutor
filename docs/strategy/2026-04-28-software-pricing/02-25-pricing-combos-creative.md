# 30 Pricing Combo Creative — ELAB Tutor Software-Only

Data: 2026-04-28. Cost SW-only steady-state 100 scuole: €386/scuola/anno.
Componenti cost: backend €216 + dev amortizzato €120 + supporto €50.

Margin formula: `margine % = (revenue - €386) / revenue * 100`
Break-even formula: `scuole minime = costi_fissi_annui / (revenue - cost_var)`

## CATEGORIA A — SUBSCRIPTION PER-SCHOOL (FORMULE 1-4)

### COMBO 1 — ELAB Light €290/scuola/anno

Base entry tier minimal.

- **Features incluse**: Lavagna LIM core (Vol1+2+3 path) + UNLIM chat base text-only + 4 giochi didattici
- **Features escluse**: voice Isabella, vision diagnose, fumetto, dashboard avanzata, ClawBot
- **Cost backend coperto**: €216 backend = SI (revenue €290 > €216)
- **Margin steady-state**: (290-386)/290 = **-33% NEGATIVO**, perde €96/scuola
- **Pros**: prezzo psicologico bassissimo, voucher PNRR copre 30 scuole con €10K
- **Cons**: NON copre cost dev+supporto, sostenibile solo come loss-leader Y1
- **Target**: scuola elementare/media diffidente, prima volta
- **Break-even**: NON raggiungibile. Funziona solo come "trial paid" Y1 con upgrade Y2
- **Benchmark**: vicino a Wokwi Classroom basso, ben sotto MyEdu

### COMBO 2 — ELAB Standard €490/scuola/anno

Sweet spot funzionale.

- **Features incluse**: Light + UNLIM Voice Isabella + Vision diagnose
- **Cost coperto**: SI con margine
- **Margin**: (490-386)/490 = **+21%** = €104/scuola
- **Pros**: copre tutti i cost, voce italiana = differenziatore principale
- **Cons**: nessuno significativo
- **Target**: scuola media/superiore mainstream
- **Break-even 100 scuole**: €49.000 revenue, costo €38.600, profit €10.400
- **Benchmark**: sotto Proteus edu (€800-2500), sopra Tinkercad free

### COMBO 3 — ELAB Pro €790/scuola/anno

Tier completo.

- **Features incluse**: Standard + Fumetto sessione + Dashboard docente full + 4 giochi
- **Margin**: (790-386)/790 = **+51%** = €404/scuola
- **Pros**: margine sano, prodotto completo

- **Target**: scuole "innovative", istituti tecnici STEM
- **Break-even 50 scuole**: €39.500 revenue copre costi 100% + buffer
